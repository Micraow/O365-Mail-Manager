# O365-Mail-Manager
用Python编写的代码示例，与O365连接并查阅邮件，查看日历等。

**欢迎Pull requests和Issues!**

这是我第一个独立的项目，希望大家支持。

程序很简单，通过我的应用ID和机密连接到你的账号，并操作，运用了一个模板 [O365](https://github.com/O365/python-o365 "O365").
如果你要使用，请：
1.  `pip install O365`
2.准备一个微软账号
